1. Описание задачи

Какую проблему решает бот - бот помогает оптимизировать задачу выставления тасок, хранит дедлайны по задачам и улучшает планирование

Почему выбрали именно эту задачу - в будущем я бы хотела внести доработки и применять его в работе своей реальной команды

2. Промпт 

Создай Telegram-бота на Python с использованием библиотеки python-telegram-bot.

Функционал бота:
- Принимает задачи в текстовом виде
Организует их по приоритетам
Отправляет напоминания
Формирует списки дел
- Команды могут быть следкющие: /newtask {тело задачи} - создает новую задачу, после этого заправшивает дедлайн на выполнение,  /alltasks - вывод всех назначенных тасок с их приоритетами (id задачи, тело задачи, приоритет, дедлайн) , /completetask {id} - закрытие задачи и убрать из списка, /todo  {список id} - создать список дел (чеклист) по текстам задачи, обрезать по максимальной длине символов
- Храни информацию о задачах, отправляй уведомления за 1 день до назначенного дедлайна 

Требования:
- Бот должен быть простым и понятным
- Код должен быть хорошо прокомментирован
- Использовать файл для хранения данных (JSON или SQLite)
- Добавить обработку ошибок

Создай:

1. Файл bot.py с кодом бота
2. Файл requirements.txt с зависимостями
3. Файл README.md с инструкцией по запуску
4. Файл .env.example для примера конфигурации

Вынужденные улучшения:

Получила ответ: Ошибка связана с несовместимостью версий библиотеки python-telegram-bot. В вашем коде используется версия 20.x, но похоже, что у вас установлена более старая версия.
Заменила содержимое requirements.txt на:

и переустановила зависимости

- Но это не решило проблему, бот запустился, но отвалился при добавлении новой таски, вынесла логи из Курсора в GPT и полностью заменила bot.py, после чего бот стал корректно отрабатывать

Промпт не меняла, функционал меня полностью устроил 

3. Основные библиотеки : 

- python-telegram-bot[job-queue]
     
Это основная библиотека для работы с Telegram Bot API, помимо этого эта библиотека: официально рекомендована Telegram, асинхронная (async/await) и активно поддерживается

- python-dotenv

Загрузка переменных окружения из .env файла, безопасное хранение чувствительных данных (токен бота)

- datetime (встроенная)

Работа с датами и временем, расчет дедлайнов и напоминаний
